→ Moving Average: MA50 ve MA200 hesaplama
→ Crossover tespiti (MA50 > MA200 = bullish)

→ Fiyat sapması: current_price / 5yr_avg
# MA Crossover df['MA50'] = df['Close'].rolling(50).mean() df['MA200'] = df['Close'].rolling(200).mean() # RSI def calc_rsi(series, period=14): delta = series.diff() gain = delta.clip(lower=0).rolling(period).mean() loss = -delta.clip(upper=0).rolling(period).mean() return 100 - (100 / (1 + gain/loss))

F4
Çoklu Sinyal Skorlama
◎ Planlanan
Tek metrik yerine 3 indikatörün birleşimi ile weighted scoring sistemi.
○ MA trend → +1 / -1
○ RSI oversold/overbought → +1 / -1
○ Fiyat sapması (undervalued/overvalued) → +1 / -1
○ Score ≥ +2 → BUY · Score ≤ -2 → SELL · diğer → HOLD
def score_asset(ma_signal, rsi_val, price_dev): score = 0 if ma_signal == "bullish": score += 1 if rsi_val < 30: score += 1 elif rsi_val > 70: score -= 1 if price_dev < 0.85: score += 1 elif price_dev > 1.15: score -= 1 return "BUY" if score>=2 else ("SELL" if score<=-2 else "HOLD")

// Codex ile başla
Hazır Codex Prompt'ları
01 Create a Python module that downloads 5-year daily OHLCV data for a list of 100+ financial assets using yfinance, cleans missing values, and stores them in a normalized pandas DataFrame with columns: asset, date, open, close, high, low, volume.
02 Build a technical indicator calculator in Python using pandas_ta that computes MA50, MA200, RSI(14), and price deviation from 5-year average for each asset. Return a signal: BUY if MA50>MA200 and RSI<30, SELL if MA5070, otherwise HOLD.
03 Create a FastAPI backend with an endpoint /signals that fetches the latest price for each asset, recalculates all indicators, applies a scoring system (+1/-1 for each of 3 signals), and returns a JSON list of {asset, score, signal} objects.
04 Build a Streamlit dashboard that shows a table of 100+ assets with their BUY/SELL/HOLD signals, color-coded rows, an RSI gauge chart, and a candlestick chart with MA50 and MA200 overlays for any selected asset.
05 Implement a walk-forward backtest using vectorbt: train the signal model on 2019–2022 data, generate signals for 2023, compare predicted vs actual price direction, and output accuracy rate, Sharpe ratio, and max drawdown vs buy-and-hold benchmark.
